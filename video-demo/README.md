# CyVix demo film

This folder contains a standalone HyperFrames composition for a polished hackathon demo video.

## Style

- Mood: high-energy, technical
- Canvas: light
- Visual language: Soft Pop Neo-Brutalism

## Scene plan

1. Hero title and live product framing
2. Problem statement and data-to-decision workflow
3. Live app interaction and AI response
4. Architecture and Google Cloud stack
5. Closing impact and live URL

## Timing

- Total duration: 108 seconds
- Designed to stay well under the 3 minute limit
